Проект представляет собой платформу для загрузки пользователем видео и анализа эмоций. Пользователи могут загружать видео, после чего система проводит анализ эмоций на основе содержимого видео.
Проект написан с использованием следующих технологий(фронт):
- JavaScript (для клиентской части)
- PHP (для серверной части)
- HTML
- CSS
Проект написан с использованием следующих технологий(бэк):
- Python
- Open Source
- TensorFlow
- CPU Sphinx
- Torch
 main.py - запуск веб приложения и подгрузка данных
server.py - инстализация сервера и запуск моделей обработки аудио и видео файлов
noiser.py - ИИ модель шумоподавления
spliter.py - Модель разделение дикторов в файле
cliper.py - Создание аудиофайлов с отдельными репликами для каждого спикера
classifier.py - модель вербальной оценки 
audio_to_text.py - модель преобразования аудио в текст
semantic.py - модель сематической оценки речи
